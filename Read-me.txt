What is this project about ? 

This webpage was develped according to the assignment requirements, to delivery a well design multi-page and responsive platform having contents about the Maori culture, which has video, images, texts sections, and a form for contact and payment.

Technologies used:
HTML5, CSS3, Bootstrap, JavaScript and jQuery


 -----------------------------------
|Guideline to setup the application |
 -----------------------------------

1 - This platform was built in a IDE called Visual studio code, however, if necessary any further development another IDE could be used as well.
2 - As this platform was built using a CSS3 and jQuery framework, is required to insert the CDN link into the <head> </head> tags. 
3 - The version used for jQuery and Bootstrap
The document has the following folders:
CSS/style.css (containing all of the code having the both pages design)
images/pexels-josiah-farrow-3396660.jpg (picture used as background)
JS/script.js (containing all the validations for the forms)
index.html (html for the first page)
indextwo.html (html for the second page)
